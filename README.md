# ARFurniture-Placement-Tool-for-Home-Interior-Design

This project explores the development and evaluation of an Ex
tended Reality (XR) tool designed for home interior design. The
aim is to assess whether XR technology can improve user experi
ence, design accuracy, and client engagement in residential space
planning. Our methodology combined literature review and hands
on prototyping. A Unity-based XR prototype was developed to test
the furniture placement in a room captured by using the LiDAR
enabled camera.
The application can significantly improve spatial visualization
and allows users to experiment with furniture in real time. It can
further give higher satisfaction and reduced decision-making time
compared to traditional 2D planning tools. In conclusion, XR
enhances the design process by offering an immersive, intuitive,
and efficient medium for interior planning. These findings support
broader adoption of XR in interior design workflows.
